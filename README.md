# Mazia

Mazia is a modern AI Call Agent web application designed to manage and monitor intelligent voice assistants with clarity and efficiency. The interface is clean, fast, and professional, focusing on a smooth user experience.

Built with React + Vite, styled using Tailwind CSS, and designed for a smooth, professional user experience.


## Tech Stack

React

Vite

Tailwind CSS

### Installation

# Clone the repository

git clone https://github.com/Itsnotxarwa/mazia.git

## Navigate to the project folder

cd mazia

### Install dependencies

npm install

#### Run the development server

npm run dev


Open http://localhost:5173 in your browser to view the website.
